Open-Shell

4.3.1 版 – 通用版本

什麼是 Open-Shell?

Open-Shell™ 是增強 Windows 可用性的工具集。 它有可自訂化的 [開始] 功能表和 [開始] 按鈕，加入 Windows 檔案總管的工具列和支援各種較小的功能。


系統需求

Open-Shell 能在 Windows 7, Windows 8, Windows 8.1, Windows Server 2008 R2, Windows Server 2012 和 Windows Server 2012 R2 動作。 同時支援 32 和 64 位元 (以相同安裝程式)。 某些 [開始] 功能表的面板需要 啟用 Aero 佈景主題。 其它至少需要 Basic 佈景主題。


元件


Open-Shell 有 3 個主要元件:
  • Open-Shell Menu
  • Classic Explorer
  • Classic IE

解除安裝

您可以從 控制台 -> 程式和功能 解除安裝 Open-Shell。 另一種方式是 再次執行安裝程式並選擇 [移除]。
可能需要登出以完成程序。
