Classic IE


Classic IE ist ein kleines Plugin für den Internet Explorer mit folgenden Merkmalen:
  • Fügt in der Titelleiste eine Beschriftung ein, so dass der vollständige Titel der Seite gesehen werden kann
  • Zeigt die Sicherheitszone in der Statusleiste
  • Zeigt den Download-Fortschritt in der Statusleiste

Installation

Wenn der Internet Explorer nach der Classic IE Installation zum ersten Mal ausgeführt wird, kann es sein, dass eine Eingabeaufforderung über die Installation des neuen Add-on ClassicIEBHO angezeigt wird und das Aktivieren des Add-on bestätigt werden soll. Klicken Sie auf die Schaltfläche Aktivieren. Wenn diese Eingabeaufforderung nicht angezeigt wird, gehen Sie auf Extras -> Add-Ons verwalten und stellen Sie sicher, dass ClassicIEBHO aktiviert ist. Nach der Aktivierung des Add-On muss der Internet Explorer zur endgültigen Plugin Aktivierung neu gestartet werden.

Einstellungen

Die Einstellungen können über Extras -> Classic IE Einstellungen oder über das Start-Menü aufgerufen werden. Über die Einstellungen kann die Farbe und die Schriftart verändert werden und ausgewählt werden welche Informationen auf der Statusleiste angezeigt werden.


Sie können die Grundeinstellungansicht auswählen oder alle Einstellungen anzeigen lassen. Bewegen Sie den Mauszeiger über die einzelnen Einstellungen, um eine Beschreibung der jeweiligen Option zu sehen. Verwenden Sie das Suchfeld, um eine Einstellung nach dem Namen zu suchen.
Jede Einstellung hat einen Standardwert. Der Standardwert kann konstant sein oder kann von den aktuellen Systemeinstellungen abhängen. Sobald Sie eine Einstellung ändern, wird diese "modifiziert" und fett dargestellt. Um die Einstellung auf den Standardwert zurückzusetzen, muss auf die jeweilige Option mit der rechten Maustaste geklickt werden.

Sie können die Einstellungen in einer XML-Datei speichern und später wieder laden. Um diese Funktion verwenden zu können, klicken Sie auf die Schaltfläche Speichern. Dort können Sie auch alle Einstellungen auf ihren Standardwert zurücksetzen.

Zum Übernehmen der Einstellungen müssen Sie OK drücken. Zum Aktivieren der neuen Einstellungen muss der Internet Explorer neu gestartet Werden.

Administrative Einstellungen

Die Einstellungen sind für den aktuellen Benutzer und werden in der Registry gespeichert. Standardmäßig kann jeder Benutzer seine gesamten Einstellungen bearbeiten. Ein Administrator kann bestimmte Einstellungen sperren, so dass diese von keinem Benutzer bearbeiten werden können. Dies funktioniert dadurch, indem Sie die jeweilige Einstellung zu dem HKEY_LOCAL_MACHINE\SOFTWARE\OpenShell\ClassicIE Registrierungsschlüssel hinzufügen.

Vielleicht möchten Sie auch nicht eine Einstellung sperren, sondern nur den Ausgangswert überschreiben. Dann fügen Sie zu dem Namen des jeweiligen Registrierungswerts die Ergänzung "_Default" hinzu.

Der einfachste Weg um den Namen des jeweiligen Registrierungswertes herauszufinden, ist eine Einstellung zu verändern und danach im Registrierungsschlüssel HKEY_CURRENT_USER\Software\OpenShell\ClassicIE\Settingsnachzuschauen.
Manchmal möchten Sie vielleicht eine Einstellung unverändert auf ihren Standardwert belassen, ohne den Standardwert zu kennen. Dann erstellen Sie einen DWORD-Wert und setzen Sie ihn auf den Wert 0xDEFA.
