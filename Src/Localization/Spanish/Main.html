Open-Shell

Versión 4.3.1 – lanzamiento general

¿Qué es Open-Shell?

Open-Shell™ es una colección de mejoras de usabilidad para Windows. Tiene un menú y un botón Inicio personalizables, agrega una barra de herramientas al Explorador de Windows y admite una variedad de características más pequeñas.


Requisitos del sistema

Open-Shell funciona en Windows 7, Windows 8, Windows 8.1, Windows Server 2008 R2, Windows Server 2012 y Windows Server 2012 R2. Son compatibles tanto las versiones de 32 como de 64 bits (el mismo instalador funciona en ambas). Algunas máscaras para el menú inicio requieren que se habilite el tema Aero. Otras requieren al menos el tema Básico.


Componentes


Open-Shell tiene tres componentes principales:
  • Open-Shell Menu
  • Classic Explorer
  • Classic IE

Desinstalación

Puede desinstalar Open-Shell desde Panel de control -> Programas y características. Otra forma es  ejecutar de nuevo el programa de instalación y elegir "Quitar".
Puede que para completar el proceso tenga que cerrar sesión.
